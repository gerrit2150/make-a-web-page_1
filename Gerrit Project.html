learnings from lesson three

stay away from repetition
One, it avoids errors - doing something again does not mean you do it correctly again. Two, it saves time, as you write less code, look for errors in fewer places and navigate and comprehend quickers as there is less code. css files are a great mean to avoid repetition when it comes to style.
Cascading Style Sheets
Cascading Style Sheets (css) allow to control the style of related html elements. class names are assigned to html elements. Style for each class name is determined in the css file. This simplifies and avoids repetition.
structure for the programming process
  • Look for natural boxes
  • look for repeated styles and elements
  • write html
  • apply style
  • test and refine
